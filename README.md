## 📖 About Baymax

**Baymax** is an AI-powered personal healthcare assistant inspired by Disney's iconic healthcare robot.

The application combines **Artificial Intelligence**, **interactive 3D visualization**, and **health analytics** to provide users with a modern and engaging digital healthcare experience.

Users can monitor vital signs, visualize the human body in 3D, generate detailed health reports, review historical health data, and interact with an AI-powered health assistant—all within a beautiful, intuitive interface.

# 🚀 Getting Started

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/HackSparrow-11/Baymax.git
```

## 2️⃣ Navigate to the Project

```bash
cd Baymax
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Start the Development Server

```bash
npx expo start
```
